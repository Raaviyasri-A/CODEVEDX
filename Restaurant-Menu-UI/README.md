# Restaurant Menu UI/UX Design

## Project Name
Annam South Kitchen

## Description
A mobile restaurant menu application designed for South Indian cuisine. The design focuses on clean navigation, food categorization, and an engaging user experience.

## Features
- Search dishes
- Category filters
- Popular dishes section
- Food detail screen
- Add to Cart button
- Interactive prototype
